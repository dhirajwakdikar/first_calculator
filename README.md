# 🧮 First Calculator

First Calculator is a simple and modern web-based calculator developed using Flask. The application allows users to perform basic arithmetic calculations through an interactive and responsive user interface.

## Features

- Addition, Subtraction, Multiplication, and Division
- Clean and modern design
- Responsive layout
- Flask-powered backend
- Real-time calculation using JavaScript Fetch API
- Error handling for invalid expressions

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- JSON

## Project Preview

This calculator features a dark-themed interface with a gradient background, smooth button animations, and a user-friendly design.

## Learning Objectives

This project helped me learn:

- Flask routing and templates
- Frontend and backend integration
- Handling API requests with Fetch
- Working with JSON data
- Organizing Flask project structure
